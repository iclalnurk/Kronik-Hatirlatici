# Kronik-Hatırlatıcı

**Kronik-Hatırlatıcı**, kullanıcıların günlük ilaçlarını ve rutin sağlık hatırlatmalarını takip edebilmelerini sağlayan mobil uygulamadır. Uygulama; kullanıcıların ilaç, doz ve kullanım zamanlarını belirlemesine, bu bilgilere göre anlık bildirimler almasına ve geçmiş kullanım verilerini takip etmesine olanak tanır.

---

## 📌 Özellikler

- 📅 Kullanıcıya özel **ilaç planı oluşturma**  
- ⏰ **Bildirim tabanlı hatırlatmalar**  
- 💊 İlaç adı, doz ve kullanım saatleri takibi  
- 🧠 Kullanıcı dostu ve sade arayüz

---

## 🚀 Kullanılan Teknolojiler

Bu proje aşağıdaki teknolojiler kullanılarak geliştirilmiştir:

- **React / React Native** — Mobil uygulama geliştirme  
- **JavaScript** — Uygulama mantığı  
- **Firebase** — Gerçek zamanlı veritabanı ve bildirim yönetimi
